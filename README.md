# Trabalho de ED

Neste trabalho vamos abordar um problema com estruturas de dados aprendidas na cadeira de ED

O objetivo deste trabalho é criar um jogo composto por um labirinto onde há um ou mais jogadores(bots ou não) e quem chegar ao centro primeiro ganha

# Organaização

Para facilitar o trabalho foram desenvolvidas várias interfaces em que cada elemento do grupo implementa um certo numero delas
